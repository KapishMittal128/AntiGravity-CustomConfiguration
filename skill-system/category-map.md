# Antigravity Skill Taxonomy

When creating new skills, they should ideally map to one of the following core categories to maintain a balanced library.

### 1. Core Engineering (`architecture/`, `refactoring/`, `debugging/`)
System design, refactoring, code quality, and debugging workflows.

### 2. Frontend / Website (`frontend/`, `react/`, `ui/`)
Component design, state management, accessibility, and modern frontend frameworks.

### 3. Backend / API (`backend/`, `api/`, `auth/`)
API design (REST/GraphQL), error handling, authentication, and server-side logic.

### 4. Database / Data (`database/`, `sql/`, `caching/`)
Schema design, query optimization, connection management, and data migrations.

### 5. App / Product (`product/`, `onboarding/`)
Product design thinking, user flows, MVP scoping, and business logic mapping.

### 6. Design / UX (`design/`, `ux/`)
Interface critique, design systems, and interaction patterns.

### 7. Integrations / Automation (`webhooks/`, `automation/`)
Third-party APIs, webhook handling, and automation scripts.

### 8. Infra / Deploy (`deploy/`, `ci-cd/`, `docker/`)
Deployment workflows, CI/CD pipelines, Docker, and infrastructure as code.

### 9. Research / Analysis (`research/`, `forensics/`)
Codebase analysis, competitor research, and deep technical summaries.

### 10. Growth / SEO / Launch (`seo/`, `copywriting/`, `launch/`)
Launch preparation, marketing copy, content strategy, and technical SEO.

### 11. Security & Production Review (`security/`, `audit/`, `scale/`)
Concurrency safety, race conditions, IDOR prevention, abuse resistance, and scaling bottlenecks.

### 12. AI / ML / LLM (`ai/`, `rag/`, `embeddings/`)
RAG pipelines, vector databases, embedding strategies, LLM orchestration, prompt engineering, and AI agent architectures.

### 13. Testing & Quality (`testing/`, `qa/`)
Unit, integration, and E2E testing patterns. TDD workflows, mocking strategies, and coverage analysis.
